# AstroSage Call Center Analysis

## Project Overview

This project analyzes AstroSage call center operations using Microsoft Excel. The objective was to evaluate business performance, customer satisfaction, revenue trends, and guru performance through data-driven insights.

## Objectives

* Analyze call volume trends
* Measure revenue performance
* Evaluate customer satisfaction
* Assess guru performance
* Provide strategic business recommendations

## Dataset

The dataset contains call center operational data including customer interactions, call details, ratings, revenue metrics, and consultant performance information.

## Analysis Performed

* Data Cleaning and Validation
* KPI Analysis
* Call Volume Analysis
* Revenue Analysis
* Customer Satisfaction Analysis
* Guru Performance Analysis

## Key Insights

* Identified call volume patterns and peak periods
* Evaluated revenue contribution across different segments
* Measured customer satisfaction trends
* Compared consultant performance metrics
* Generated actionable business recommendations

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Dashboarding
* Data Analysis
